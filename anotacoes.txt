/// Efeito zebrado em tabela:

para fazer o efeito zebrado usa-se o nth-child()
entre os parentes coloca de quantas em quantas linhas é para pular a Cor

EXEMPLO ==>

tbody > tr:nth-child(2n){
            background-color: lightgrey;
        }

Neste exemplo a cor alterna de 2 em 2 linha(Efeito normal de uma tabela zebrada)

Outra parametro que pode ser usado entre os parenteses é o "odd" que significa impar ou o "even" que significa par

EXEMPLO ==>
    tbody > tr:nth-child(odd) {
        background-color: lightgrey;
    }

    OU
    
    tbody > tr:nth-child(even) {
        background-color: lightgrey;
    }

// SCOPO

Scop é feito somente em TH
Ele serve para mostrar em que sentido esta os dados daquele TH

===> scope="col" => Usado quando o scope (Dados) esta em na coluna
===> Scope="row" => Usado quando o scope (Dados) esta em linha

===> scope="colgroup" => Usado em celulas mescladas quando os dados está em colunas e existe mais de uma coluna com dados 

===> scope="rowgroup" => Usado em celulas mescladas quando os dados esta em linhas e existe mais de uma linha com dados

/// IFRAME

Tamanho padrão do iframe => 300 X 150
Para estilizar o iframe da para fazer da para estilizar na tag com os tags de tamanho sem utilizar o style
No css da para estilizar e sempre vai ser soberano ao estilo incluso no iframe

O frameborder é responsável pelas bordas e pode ser estilizado por css tbm
A tag scolling inclui ou retira o scroll do iframe, porem as vexes não funciona nos navegadore


Sempre é bom colocar um aviso ou um link entre as tags de iframe pois existe navegadores que não são compativeis e assim quando não aparecer o iframe aparece o link e ou a mensagem para onode você quer encaminhar

Exemplo ===>

<p>Acessando o site do 
    <iframe src="https://www.cursoemvideo.com" frameborder="0">
        <a href="https://www.cursoemvideo.com" target="_blank">Curso em Vídeo</a>  <!--Este link só aparece se o navegador não for compativel-->
    </iframe> para aprender programar
</p>


=============== /// NAVEGAÇÃO POR IFRAME

Para uzar a navegação no iframe é preciso declarar um name para ele
Apos dedclarar o name você chama ele no link que quer utilizar para acessar o iframe

Exemplo ==>

<iframe name="frame" src=""></iframe>

Para eu incluir um conteudo neste iframe basta eu usar o name "frame" no target do link e o conteudo vai surgir dentro deste iframe

Exemplo Usando Lista ==>

<ul>
    <li><a href="./pgExtras/pg001.html" target="frame">Primeira Página</a></li>
    <li><a href="./pgExtras/pg002.html" target="frame">Segunda Página</a></li>
    <li><a href="./pgExtras/pg003.html" target="frame">Terceira Página</a></li>
</ul>




=========== /// Conteúdo no iframe por código


Para incluir conteudo por codigo no iframe basta usar a tag srcdoc

Exemplo ==>

<iframe name="frame" srcdoc="<h1 style='text-align: center;'>Escolha Umas Das Opções Acima</h1><p style='text-align: center;'>Lorem ipsum dolor sit, amet consectetur adipisicing elit. Non, dolorum sunt. Illo odit eius accusamus veritatis quas amet molestiae. Voluptates temporibus numquam quis quidem, aut dolorem fugit quasi architecto ipsam omnis quia neque magnam officiis</p>">
</iframe>




=========== /// Tornar o iframe Seguro

sandbox="sandbox"
referrerpolicy="no-referrer"

Esses dois atributos juntos liga o maximo de segurança no iframe

Exemplo ==>

<iframe src="pgExtras/pg004-form.html" frameborder="0" width="500" height="150" sandbox="sandbox" referrerpolicy="no-referrer">
    Seu navegador não é compativel com iframe...
</iframe>

Se quiser liberar por exemplo um iframe local e um formulário para captura usa-se os atributos:

sandbox="allow-same-origin" => Libera o iframe para conteudo interno
sandbox="allow-form" => libera o formulário
sandbox="allow-script" => libera os scripts que tbm são usados no formulário

Exemplo de uso ==>

 <iframe src="pgExtras/pg004-form.html" frameborder="0" width="500" height="150" sandbox="allow-same-origin allow-forms allow-scripts" referrerpolicy="no-referrer">
   Seu navegador não é compativel com iframe...
</iframe>
